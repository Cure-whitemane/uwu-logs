How to start logging

  • Type /combatlog in game (before anything happens, not after)
  • Find logs file in game's folder: WoW/Logs/WoWCombatLog.txt
  • Archive it (supported formates: .7z or .zip)
  • Upload above

Additional info

  • Combatlog is logged into 1 file always
  • After upload, don't forget to delete WoWCombatLog.txt to keep it fresh
  • (or rename the file, if you want to keep it)
  • Use weakauras to never forget to start logging
  • You can /reload mid pull with this wa and it ll continue logging
  • Use 7zip with PPMd compression for lower archiving time and file size
  • 1GB Logs archived time: 7z 20sec Zip 80sec
  • 1GB Logs archived size: 7z 30MB Zip 90MB
